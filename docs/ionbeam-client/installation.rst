Installation
============

Requirements
------------

* Python 3.12 or later

Install
-------

.. code-block:: bash

   uv pip install ionbeam-client

Or with pip:

.. code-block:: bash

   pip install ionbeam-client

Development Installation
------------------------

.. code-block:: bash

   git clone https://github.com/ecmwf/ionbeam.git
   cd ionbeam/ionbeam-client
   uv pip install -e .[dev]
